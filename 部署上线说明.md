## 推荐部署方式

### 方式一：Netlify

适合新手，最简单。

操作方式：

1. 打开 Netlify
2. 选择 Add new site
3. 上传 `output/测试主题` 文件夹
4. Netlify 会返回一个可以访问的链接

优点：

- 简单
- 不需要写服务器
- 静态网页直接能用

### 方式二：GitHub Pages

适合以后批量化。

流程：

1. 把生成的 `index.html` 上传到 GitHub 仓库
2. 开启 GitHub Pages
3. 每个测试页面都有一个固定链接

### 方式三：Cloudflare Pages

适合更稳定地批量部署。

流程：

1. 连接 GitHub 仓库
2. 每次生成新测试网页后提交代码
3. Cloudflare 自动部署并给出链接
