# 测试页界面参考拆解

参考图：职业方向测试 H5 首屏。

## 一、整体气质

这个界面不是可爱风，也不是强营销风，而是偏“高级、理性、可信”的测试报告风格。

关键词：

- 纸张感
- 网格背景
- 深蓝黑主色
- 灰色辅助文案
- 留白多
- 信息密度适中
- 像一份认真设计过的测评

适合用在：

- 职业测试
- 人格测试
- 关系模式测试
- 自我探索测试
- 付费报告型 H5

## 二、页面结构

首屏结构可以复用成：

1. 顶部英文标签
2. 中文大标题
3. 两行副标题
4. 细分割线
5. 痛点导入文案
6. 测试价值说明
7. 四个测评维度标签
8. 深色大按钮
9. 底部免责声明

## 三、视觉细节

### 背景

使用非常浅的网格纸背景。

效果：

- 像笔记本或测评表
- 提升专业感
- 让页面不空，但不抢内容

可以用 CSS 实现：

- 底色：`#f8f9f8`
- 细线：浅灰透明
- 横竖网格：约 24px 一格

### 字体

标题：

- 粗体
- 深蓝黑
- 字号大
- 行距紧
- 居中

正文：

- 深灰
- 行距较大
- 每段之间留白明显

英文标签：

- 全大写
- 字母间距拉开
- 两边用圆点装饰

### 颜色

主色：

- 深蓝黑：`#172635`

辅助灰：

- 正文灰：`#5f6b73`
- 浅灰线：`#d9dee2`

背景：

- 米白/灰白：`#f8f9f8`

按钮：

- 深蓝黑底
- 白字
- 大面积、强行动感

## 四、文案结构

### 英文小标题

格式：

`• CAREER PATH NAVIGATOR •`

作用：

- 增加仪式感
- 让普通测试看起来更像一个产品

不同主题可替换：

- 职业：`CAREER PATH NAVIGATOR`
- 恋爱：`LOVE PATTERN NAVIGATOR`
- 人格：`PERSONALITY TYPE MAP`
- 情绪：`EMOTIONAL STATE CHECK`
- 财富：`MONEY MINDSET TEST`

### 主标题

格式：

`你现在该换职业方向，还是继续深耕？`

特点：

- 二选一
- 强痛点
- 直接指向用户当下困惑

### 副标题

格式：

`不是每一次想转行，都是真的不适合。`
`也不是每一次坚持，都值得继续耗。`

特点：

- 两句对称
- 否定常见误区
- 制造“需要测试判断”的理由

### 痛点导入

写法：

- 你有没有过这种感觉
- 看到别人转行会心动
- 但又害怕换错
- 继续坚持又怕浪费时间

作用：

- 让用户觉得“这说的不就是我吗”

### 测试价值说明

格式：

`这个测试会从 A、B、C、D 四个维度，帮你判断：你现在更适合切换赛道，还是继续深挖当前方向。`

作用：

- 解释测试不是随便玩
- 给用户一个“测了有用”的理由

## 五、四个维度标签

参考图里是：

- 职业匹配度
- 成长反馈
- 迁移能力
- 转型动机

这个模块很重要，因为它会让测试看起来更专业。

不同主题可替换：

恋爱测试：

- 情绪投入
- 身体吸引
- 长期适配
- 安全感需求

人格测试：

- 社交能量
- 情绪反应
- 行动模式
- 自我边界

财富测试：

- 赚钱动机
- 行动力
- 风险承受
- 资源整合

## 六、按钮

按钮文案：

`开始测试`

设计：

- 宽度接近整屏内容区
- 深色背景
- 白色粗体
- 高度足够大
- 不用花哨渐变

这个按钮的感觉是“正式进入测评”，不是普通小按钮。

## 七、适合加入生成器的模板参数

以后每个测试可以配置：

- `eyebrow`: 英文小标题
- `title`: 中文标题
- `subtitle_lines`: 两行副标题
- `intro_paragraphs`: 痛点文案
- `value_statement`: 测试价值说明
- `dimensions`: 四个维度标签
- `button_text`: 开始测试
- `disclaimer`: 免责声明

## 八、可复用页面骨架

```text
• ENGLISH NAVIGATOR •

中文大标题

副标题第一句
副标题第二句

——

痛点段落 1

痛点段落 2

痛点段落 3

加粗说明：这个测试会从四个维度，帮你判断……

[维度一] [维度二]
[维度三] [维度四]

[开始测试]

免责声明
```

## 九、我的复刻建议

职业类测试优先用这个风格。

恋爱类测试可以保留这个结构，但颜色稍微柔和：

- 背景仍然用网格纸
- 主色从深蓝黑换成玫瑰棕/深酒红
- 四个维度标签保持简洁
- 不要做得太卡通

这样会比纯粉色页面更像“认真测评”，更适合低价付费测试。
