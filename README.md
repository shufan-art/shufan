# 通用测试壳

这个文件夹用来批量生成低价测试 H5 页面。

以后你只需要给一个主题，比如：

- 你适合稳定上班，还是自由赚钱？
- 你在关系里，到底需要哪种爱？
- 你最近的焦虑，到底在提醒你什么？

我就可以帮你生成：

- 测试标题
- 16 道题
- 4 个结果类型
- 结果报告
- 一个可以直接打开的网页

## 使用方法

在这个文件夹里运行：

```bash
python3 generate_test.py "你适合稳定上班，还是自由赚钱？"
```

生成后的网页会出现在：

```text
output/你的测试主题/index.html
```

双击 `index.html` 就可以在浏览器里打开。

## 上线说明

现在这个版本已经能生成静态网页。

要自动部署成公网链接，需要再接入一个部署平台：

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel

接入后，流程可以变成：

`输入主题 → 自动生成网页 → 自动上传部署 → 返回可访问链接`
